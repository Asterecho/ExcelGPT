# ExcelGPT

本插件基于ExcelDna开发，无需复杂的VSTO和VBA环境，只需双击即可安装使用
**兼容Office Excel 和 WPS** （强烈建议使用金山公司开发的WPS，插件运行的更加完美！）

单元格中输入=AI()即可食用
